Hide

The map is interactive and shows climate data at different stations. The kind of data can be selected on the top right side. Data is from (the average (mean) of seven days around) the day which can be chosen on the bottom left side. Additionally the average (median) from the chosen yearspan is taken. By clicking on two stations/cells in the map, the complete yearly data of these stations can be compared in the plot below the map. That plot furthermore contains an interactive zoom at the bottom. If you have a slow internet connection, select "Station subset". More information on my blog.
